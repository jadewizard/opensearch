Документация по функциями веб-приложения open-search.


Основные функции скрипта:

- основные функции отвечают за вывод контента на страницы сайта.


Функция

Описание функции

{% include 'header.html' %}

Функция для вывода верхней части щаблона (шапки) — header.html.

{% include sidebar %}

Функция для вывода сайдбара (боковой колонки сайта)

{% include content %}

Функция для вывода контента сайта.

{{ login_message|raw }}

Функция для вывода ошибок которые появились в ходе авторизации пользователя на сайте.

{{ message|raw }}

Функция для вывода ошибок которые появились в ходе регистрации пользователя на сайте.
